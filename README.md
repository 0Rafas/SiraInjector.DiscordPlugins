## üöÄ Getting Started

### Prerequisites

*   Windows 10 or later.
*   [.NET Framework 4.8 or newer](https://dotnet.microsoft.com/en-us/download/dotnet-framework ).
*   A desktop installation of Discord.

### Installation

1.  Navigate to the [**Releases**](https://github.com/0Rafas/SiraInjector.DiscordPlugins/releases ) page.
2.  Download the latest `DiscordPluginInjector.exe` file.
3.  Place the executable in a convenient location and run it. No complex installation required!

## üìñ How to Use

Using the injector is as simple as 1-2-3:

1.  **Launch**: Run `DiscordPluginInjector.exe`.
2.  **Import**: Click the **"Import Plugin"** button and select the plugin file you want to load.
3.  **Inject**: Once imported, select the plugin from the list and click the **"Inject"** button.

That's it! The plugin will be instantly loaded into your Discord client.
